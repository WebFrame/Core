# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
|  v2.1   | :white_check_mark: |
| <= v2   | :x:                |

## Reporting a Vulnerability

In case you find vulnerability, please make sure to open an [issue](https://github.com/WebFrame/Core/issues) so we can fix it as soon as possible.
